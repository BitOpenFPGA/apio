================================
Welcome to apio's documentation!
================================

.. image:: resources/images/apio-logo-mini.png


Experimental open source micro-ecosystem for **open FPGAs**. Based on `platformio <https://github.com/platformio/platformio>`_. Apio is a **multiplatform toolbox**, with static pre-built packages, project configuration tools and easy commands to verify, synthesize, simulate and upload your **verilog** designs.

Apio is used by `Icestudio <https://github.com/FPGAwars/icestudio>`_.

**Source code**: https://github.com/FPGAwars/apio
